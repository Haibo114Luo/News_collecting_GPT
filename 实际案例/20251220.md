# 2025.12.19
## 美国
#### 1.
- **日期**：报道2025-12-18 / 事件2025-12-18
- **地区**：美国（PJM覆盖区：中大西洋/中西部，13州+华盛顿特区）
- **类别**：政策收紧
- **事件**：（12月18日《美国联邦能源监管委员会 FERC》公告）FERC要求美国最大电网运营商PJM就“与发电设施共址（co-located）的AI数据中心等大负荷”建立更透明的接入与输电服务规则，并认定其现行tariff在共址负荷相关条款上“缺乏清晰一致性”，要求修订并于2026-01-19前提交进展报告。
- **影响**：共址接入合规门槛清晰化，短期或放慢部分项目并网节奏、抑制成本转嫁争议。
- **来源**：https://www.ferc.gov/news-events/news/ferc-directs-nations-largest-grid-operator-create-new-rules-embrace-innovation-and-protect-consumers

#### 2.
- **日期**：报道2025-12-19 / 事件2025-12-18
- **地区**：美国（PJM市场）
- **类别**：市场/统计数据
- **事件**：（12月19日《Utility Dive》报道）在FERC公开会议上，多位委员对PJM最新容量拍卖未达到可靠性目标提出警示。PJM称可通过更严格的大负荷核查、下月发布的新负荷预测等因素缩小缺口；报道同时指数据中心负荷增长与高电价压力叠加，已成为监管与市场焦点之一。
- **影响**：容量缺口与高价信号或推高电力成本预期，影响数据中心选址与用电合同条款。
- **来源**：https://www.utilitydive.com/news/ferc-pjm-reliability-data-center-large-load/808352/

#### 3.
- **日期**：报道2025-12-18 / 事件2025-12-18
- **地区**：美国（密歇根州—Saline Township/Washtenaw County）
- **类别**：政策支持
- **事件**：（12月18日《密歇根州公共服务委员会 MPSC》公告）MPSC有条件批准DTE Electric的特殊供电合同，为Saline Township拟建数据中心供电（1,383MW）。监管条件包括：DTE需承担无法向Oracle子公司Green Chile Ventures回收的供电成本；在极端负荷削减时需优先削减数据中心负荷；并设置19年最低合同期、80%最低计费需求与最高10年终止赔付等条款；MPSC提及对其他用户净受益约3亿美元。
- **影响**：为超大负荷项目“先上车”开绿灯但强化兜底约束，或成他州大负荷合同监管范本。
- **来源**：https://www.michigan.gov/mpsc/commission/news-releases/2025/12/18/mpsc-approves-dte-electric-energy-contracts-for-data-center

#### 4.

- **地区**：美国（佐治亚州）
- **类别**：政策支持

#### 7.
- **日期**：报道2025-12-19 / 事件2025-12-19
- **地区**：美国（多州）
- **类别**：政策收紧
- **事件**：（12月19日《The Verge》报道）美国多地针对超大数据中心项目的反对行动在2025年显著增加，报道援引跟踪数据称有大量项目被叫停或延后，并提到电价、用水、噪音与污染等议题推动两党地方层面收紧态度；同时出现限制税收优惠、要求更强披露/约束的新立法动向。
